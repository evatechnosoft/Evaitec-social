## PORTFÖY ÖGELERİ (3 Adet)

### Portfolio 1: E-Car Dashboard
**Başlık:** Real-time EV Monitoring App — Flutter + BLE
**Kategori:** Mobile Development

**Açıklama:**
A cross-platform mobile application for monitoring electric vehicle systems in real-time via Bluetooth Low Energy.

Key features:
• Connects to Votol EM150 and Fardriver ND72450 motor controllers
• Reads battery data from JK, JBD, and DALY BMS units
• Tracks energy consumption via Hall sensor Coulometer
• Custom BLE protocol handling (B5-5B, AA-55, 3A frames)
• Real-time dashboard with speed, battery, temperature, and range data

Tech stack: Flutter, Dart, BLE, Custom Protocol Parsers

This was built for our own custom electric vehicle — solving a real problem we encountered during the build process.
